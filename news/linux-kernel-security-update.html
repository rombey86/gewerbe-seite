Neues Sicherheitsupdate für Linux Kernel behebt kritische Schwachstelle

Ein neues Sicherheitsupdate für den Linux Kernel behebt eine kritische Schwachstelle, die es Angreifern ermöglichen könnte, Schadcode auszuführen. Es wird empfohlen, das Update so schnell wie möglich zu installieren.
